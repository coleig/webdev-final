# Wikipedia Search
## A final exam for web development featuring a wikipedia api call.

## Creative Components:
* Custom Fonts
* Favicon
* Some CSS Styling

## How to Install
1. Clone the repository
2. Navigate to the client directory and execute:
```
meteor npm install
meteor
```
3. Open `http://localhost:3000` to see the project running!
